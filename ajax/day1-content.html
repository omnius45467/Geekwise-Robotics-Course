Safety:

    If you want to learn how to build a robot, here are the rules

  • Respect the hardware. Never plug or unplug hardware while connected to power.

  • Never plug in hardware while connected to power.

  • Never unplug hardware while connected to power.

  • Never mistreat your robot! They will seek revenge.

Installation:

    The first step to building your robot is to download and install the programing environment or IDE(Integrated Programing Environment)

  • Visit this link to install the latest version of the Arduino IDE for your machine Arduino IDE

  • We will be sharing code over slack and github. Github is a repository system that allows you to store code that others can use.

Getting used to the IDE layout:

    The IDE will look slightly different depending on your computer, but all the main functionality will be the same.

  • First of all your code goes in the middle, like a normal text editor.

  • Next when the code is ready to be sent to the board, click the verify button in the top left.

  • Before you can transfer the code to the computer chip you have to make sure a couple things have been selected properly.

    • Select the board under...

      Tools/Board/"Arduino Board"

    • Check to see what Serial Port your Arduino is connected to, you can do this by clicking...

      Tools/Port/Serial Ports/"system specific port"

    • To verify that you have a connection you can look at the bottom right hand corner of the IDE.

  • The way you get the code onto the board is by click the upload button next to the verify button.

  • You can see that the code has been uploaded by the black console at the bottom of the IDE.

This is where things get a little tricky

  • Arduino Basic Code Template

  • The way you set variables

                        
// These constants won't change:
const int analogPin = A0;    // pin that the sensor is attached to
const int ledPin = 13;       // pin that the LED is attached to
const int threshold = 400;   // an arbitrary threshold level that's in the range of the analog input

  • Open a basic code example by clicking File/Examples/Basic/Blink

    This is the basic code for one of your first programs

                        
void setup() {
// initialize digital pin 13 as an output.
pinMode(13, OUTPUT);
}

// the loop function runs over and over again forever
void loop() {
digitalWrite(13, HIGH);   // turn the LED on
delay(1000);              // wait for a second
digitalWrite(13, LOW);    // turn the LED off
delay(1000);              // wait for a second
}

  • The Arduino if statement 

                        
// These constants won't change:
const int analogPin = A0;    // pin that the sensor is attached to
const int ledPin = 13;       // pin that the LED is attached to
const int threshold = 400;   // an arbitrary threshold level that's in the range of the analog input

void setup() {
// initialize the LED pin as an output:
pinMode(ledPin, OUTPUT);
// initialize serial communications:
Serial.begin(9600);
}

void loop() {
// read the value of the potentiometer:
int analogValue = analogRead(analogPin);

// if the analog value is high enough, turn on the LED:
if (analogValue > threshold) {
digitalWrite(ledPin, HIGH);
}
else {
digitalWrite(ledPin, LOW);
}

// print the analog value:
Serial.println(analogValue);
delay(1);        // delay in between reads for stability
}
