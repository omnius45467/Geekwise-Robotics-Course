Code Review:

Let's go over the basic parts of Arduino code again

  • Basic structure

    Like in the picture what makes up the basic structure of an Arduino program?

  • Constant Variables

    These go near the top of the program

                            
// These constants won't change:
const int analogPin = A0;
const int ledPin = 13;
const int threshold = 400;

  • If Statements

    Of course this bit of code needs to be inside a loop 

                            
// if the analog value is high enough, turn on the LED:
if (analogValue > threshold) {
digitalWrite(ledPin, HIGH);
}
else {
digitalWrite(ledPin, LOW);
}
